# Study Hours Tracker 📊

This project is a simple Python application that tracks weekly study hours and visualizes them using Matplotlib.

## 🔧 Features
- Input daily study hours
- Calculate average study time
- Generate a line chart visualization

## 🛠 Technologies
- Python
- Matplotlib
